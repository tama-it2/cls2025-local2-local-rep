## 天津市（Tianjin）

- 中国北方的重要直辖市，经济与港口物流发达  
- 历史上是中外文化交流的重要窗口，有多个历史风貌区  
- 美食丰富，有名的狗不理包子、煎饼果子、耳朵眼炸糕等  
- 教育资源丰富，有南开大学、天津大学等重点高校  
- 正在推进智能制造和航天科技发展
